BOT LOGIC REFERENCE:
  Trigger BUY  : EMA20 ตัดขึ้นเหนือ EMA50 (Golden Cross) + ADX > 25
  Trigger SELL : EMA20 ตัดลงใต้ EMA50 (Death Cross) หรือ SL/TP โดน

  SL  = entry − (ATR × 1.5)   TP1 = entry + (ATR × 1.0)  R:R 1:1
  TP2 = entry + (ATR × 2.0)   TP3 = entry + (ATR × 3.0)  R:R 1:3
  Position Size = (balance × 1%) / (entry − SL)  → เสี่ยงสูงสุด 1% ต่อ trade
  Kill Switch   = daily loss > 5% → หยุดบอท 24 ชั่วโมง

  streamlit run dashboard/app.py                 # เปิด dashboard → localhost:8501
